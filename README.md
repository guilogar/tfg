# tfg
Repository for make final degree project

## Create a .env for environment variables
### backend/.env
```
DATABASE_URL=your database url
```

### frontend/.env
```
DATABASE_URL=your database url
```

## Database Posgresql
```
The database must be a postgresql database
```

## Start backend
```
cd backend
node .
```

## Start frontend
```
cd frontend
ionic serve
```
